# 바로가기 만들기

폰 홈 화면 + PC 바탕화면 단축 아이콘 만드는 방법. 1분 안에 끝남.

**URL**: `https://agqen.github.io/trendstock/`

---

## 📱 핸드폰 (홈 화면에 앱처럼 추가)

### Android Chrome

1. Chrome 브라우저로 `https://agqen.github.io/trendstock/` 접속
2. 우상단 **⋮** 메뉴 클릭
3. **앱 설치** 또는 **홈 화면에 추가** 클릭
4. 이름 확인 → **추가** 클릭

→ 홈 화면에 **TrendStock 아이콘** 생성. 탭하면 **풀스크린 앱처럼** 실행.

### iPhone Safari

1. Safari 로 `https://agqen.github.io/trendstock/` 접속
2. 하단 **공유 버튼** (위로 향한 화살표 □↑) 탭
3. 스크롤 → **홈 화면에 추가** 탭
4. 이름 확인 → 우상단 **추가** 탭

→ 홈 화면에 아이콘 생성.

---

## 💻 PC (바탕화면 단축 아이콘)

### Chrome

1. Chrome 으로 `https://agqen.github.io/trendstock/` 접속
2. 우상단 **⋮** 메뉴 클릭
3. **저장 후 공유** → **바로가기 만들기...**
4. 이름: `TrendStock` 입력
5. **창으로 열기** 체크박스 ✅
6. **만들기** 클릭

→ 바탕화면에 아이콘 생성. 더블클릭 시 **브라우저 메뉴 없는 전용 창**으로 실행 (앱과 비슷한 경험).

### Edge

1. Edge 로 같은 URL 접속
2. 우상단 **⋯** → **앱** → **이 사이트를 앱으로 설치**
3. 이름 입력 → 설치
4. 시작 메뉴 / 작업 표시줄에 추가 가능

---

## 🔁 매일 새 분석 확인 흐름

1. 홈 화면의 **TrendStock 아이콘** 탭
2. 자동으로 최신 데이터 로딩 (Service Worker 가 백그라운드에서 새 JSON fetch)
3. 좌상단 **≡** → **🕓 히스토리** 로 어제와 비교

---

## 🆘 새 분석 안 보이면

새 데이터가 GitHub Pages 에 올라왔는데 PWA 화면이 아직 옛 것이라면:

- 폰: 앱 닫고 다시 열기 (1~2회)
- PC: `Ctrl+Shift+R` (강력 새로고침)
- 그래도 안 되면 PWA 삭제 후 재설치

매일 자동 갱신 + 새 알람 같은 push notification 은 향후 네이티브 앱(`native-app/`) 단계에서 추가 예정.
