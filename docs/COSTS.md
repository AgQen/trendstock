# 💰 토큰 / 비용 정리

매일 자동 운영에 들어가는 돈을 작업별로 분해 정리. 모든 비용은 Anthropic API 호출 한정 (Pro 구독료와 별개).

> **환율**: $1 ≈ ₩1,400 가정.
> **단가** (2026-05 기준, $/MTok):
>   - Sonnet 4.6 (default) — input $3, output $15, cache_read $0.30
>   - Haiku 4.5 (저렴) — input $1, output $5, cache_read $0.10
>   - Opus 4.7 (고급) — input $15, output $75, cache_read $1.50

---

## 1️⃣ 일일 분석 (daily_analysis)

매일 KST 06:50 자동 실행. 가격 + 거시 + 뉴스 → 새 트렌드 5+5 + 추천 약 30건 생성.

| 모델 | 입력 토큰 | 출력 토큰 | 1회 비용 | **월 (30회)** |
|---|---|---|---|---|
| **Sonnet 4.6** (default) | ~8,000 | ~3,000 | $0.069 | **$2.07 ≈ ₩2,900** |
| Haiku 4.5 | ~8,000 | ~3,000 | $0.023 | $0.69 ≈ ₩960 |
| Opus 4.7 | ~8,000 | ~3,000 | $0.345 | $10.35 ≈ ₩14,500 |

**기본 운영비 = 월 약 3,000원.**

## 2️⃣ 모델 강화 / 회고 (reinforcement)

매 분기 (3개월) 1회. 누적 적중률 데이터를 보고 룰 가중치 + 프롬프트 자동 튜닝.

| 항목 | 토큰 | 비용 |
|---|---|---|
| 입력 (3개월치 추천 + 검증 결과 + 적중 패턴) | ~30,000 | $0.09 |
| 출력 (룰 조정안 + 새 프롬프트) | ~5,000 | $0.075 |
| **1회 합계** | | **$0.165 ≈ ₩230** |
| **연 4회** | | **₩920** |

→ 거의 무시 가능. 모델 강화는 핵심 가치인데도 비용 부담 없음.

## 4️⃣ 무료 작업 (LLM 없이)

다음은 코드 / 룰 / 외부 무료 API 만 사용 — **비용 $0**:

| 작업 | 사용 도구 | 빈도 |
|---|---|---|
| 가격 수집 (yfinance / pykrx) | 무료 API | 매일 |
| 룰 기반 등급 (펀더 + 모멘텀 + 타이밍) | 로컬 계산 | 매일 |
| 적중률 검증 (7d / 30d / 90d 알파) | 로컬 계산 | 매일 |
| 히스토리 / 차트 / Excel 다운로드 | 클라이언트 사이드 | 사용자 요청 시 |
| GitHub Pages 호스팅 | 무료 | 24/7 |
| GitHub Actions cron | 월 2,000분 무료 | 매일 ~3분 사용 |

→ **LLM API 키 미충전 상태에서도** 가격/룰 등급/적중률은 매일 자동 갱신됨. 새 LLM 분석만 생성 안 됨.
